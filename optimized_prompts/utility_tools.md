# Prompt Otimizado: Ferramentas e Utilitários

## Calculadora Avançada

```
Crie uma calculadora avançada com as seguintes especificações:

1. Funcionalidade:
   - Operações básicas (adição, subtração, multiplicação, divisão)
   - Funções científicas (seno, cosseno, tangente, logaritmo, exponencial, raiz quadrada)
   - Memória (armazenar, recuperar, adicionar à memória, subtrair da memória)
   - Histórico de cálculos com capacidade de reutilizar resultados anteriores
   - Conversão entre diferentes sistemas numéricos (decimal, binário, hexadecimal, octal)
   - Cálculo de porcentagens e juros
   - Suporte a parênteses e ordem de operações

2. Design:
   - Interface limpa e profissional
   - Esquema de cores: fundo escuro (#212121), botões em cinza (#424242), botões de operação em laranja (#ff9800), resultados em branco
   - Layout responsivo que se adapta a diferentes tamanhos de tela
   - Teclado virtual bem espaçado para facilitar o uso
   - Design inspirado em calculadoras científicas modernas

3. Dados:
   - Armazenamento de histórico de cálculos usando localStorage
   - Persistência de configurações do usuário
   - Precisão decimal configurável

4. Interação:
   - Entrada via teclado virtual ou teclado físico
   - Feedback visual ao pressionar botões
   - Animações sutis para transições
   - Modo de cópia rápida de resultados
   - Alternância entre modos (básico, científico, programador)

5. Recursos adicionais:
   - Temas alternáveis (claro/escuro)
   - Calculadora de datas (diferença entre datas, adicionar/subtrair dias)
   - Conversor de unidades integrado
   - Modo de gráficos para funções simples
   - Customização da disposição dos botões
```

## Conversor Universal de Unidades

```
Crie um conversor universal de unidades com as seguintes especificações:

1. Funcionalidade:
   - Conversão entre múltiplas categorias: comprimento, área, volume, massa, temperatura, tempo, velocidade, pressão, energia, potência, dados digitais
   - Suporte a unidades métricas, imperiais e outras unidades comuns
   - Conversão instantânea ao digitar valores
   - Precisão decimal ajustável
   - Favoritar conversões frequentes
   - Histórico de conversões recentes
   - Calculadora básica integrada

2. Design:
   - Interface limpa e funcional
   - Esquema de cores: azul (#2196f3) para elementos principais, branco para fundo, cinza para detalhes
   - Layout com seleção de categoria no topo, unidades de origem e destino abaixo
   - Organização clara das unidades por relevância
   - Design responsivo para todos os dispositivos
   - Inspirado em aplicativos de conversão profissionais

3. Dados:
   - Banco de dados completo de unidades e fatores de conversão
   - Armazenamento de favoritos e histórico usando localStorage
   - Fórmulas de conversão precisas para casos especiais (temperatura)

4. Interação:
   - Seleção rápida de unidades com pesquisa
   - Atualização em tempo real dos valores convertidos
   - Botão de inversão para trocar unidades de origem e destino
   - Cópia de resultados com um clique
   - Feedback visual durante a interação

5. Recursos adicionais:
   - Modo offline completo
   - Conversões de moedas com taxas simuladas
   - Informações contextuais sobre cada unidade
   - Temas claro/escuro
   - Exportação de tabelas de conversão
```

## Gerador de Senhas Seguras

```
Crie um gerador de senhas seguras com as seguintes especificações:

1. Funcionalidade:
   - Gerar senhas aleatórias com comprimento ajustável (8-64 caracteres)
   - Opções configuráveis para incluir letras maiúsculas, minúsculas, números e símbolos
   - Verificação de força da senha gerada
   - Copiar senha para a área de transferência com um clique
   - Histórico de senhas geradas (opcional, com aviso de segurança)
   - Geração de frases-senha (passphrase) usando palavras comuns
   - Verificação de tempo estimado para quebrar a senha

2. Design:
   - Interface minimalista e segura
   - Esquema de cores: verde (#4caf50) para elementos principais, branco para fundo, vermelho (#f44336) para alertas
   - Layout centralizado com controles intuitivos
   - Indicador visual de força da senha (barra colorida)
   - Design responsivo para todos os dispositivos
   - Inspirado em geradores de senha de ferramentas de segurança

3. Dados:
   - Algoritmo de geração de senhas verdadeiramente aleatórias
   - Dicionário para geração de frases-senha
   - Cálculo de entropia para avaliação de força
   - Nenhum armazenamento persistente por padrão (opcional)

4. Interação:
   - Sliders e toggles para configurações
   - Botão de geração proeminente
   - Feedback visual ao copiar senha
   - Atualização instantânea da força da senha
   - Opção para mostrar/ocultar a senha gerada

5. Recursos adicionais:
   - Modo de geração de múltiplas senhas
   - Dicas para criar senhas memoráveis mas seguras
   - Verificação contra senhas comuns/vazadas
   - Temporizador para limpar senha da tela
   - Geração de QR code com a senha
```
