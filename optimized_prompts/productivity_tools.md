# Prompt Otimizado: Ferramentas de Produtividade

## Gerenciador de Projetos

```
Crie um gerenciador de projetos com as seguintes especificações:

1. Funcionalidade:
   - Criar e gerenciar múltiplos projetos com título, descrição, data de início e prazo
   - Adicionar tarefas a cada projeto com título, descrição, responsável, prioridade e status
   - Organizar tarefas em quadros kanban (A fazer, Em andamento, Concluído)
   - Acompanhar progresso geral do projeto com barra de progresso
   - Definir dependências entre tarefas
   - Filtrar e ordenar tarefas por diferentes critérios
   - Gerar relatórios de progresso e produtividade

2. Design:
   - Interface profissional com tema moderno
   - Esquema de cores: azul (#2196f3) para elementos principais, cinza claro (#f5f5f5) para fundo, cores distintas para prioridades (vermelho, amarelo, verde)
   - Layout flexível com sidebar para navegação entre projetos
   - Visualização em lista e em quadro kanban
   - Design responsivo para desktop e tablet
   - Inspirado em ferramentas como Asana/Trello

3. Dados:
   - Estrutura de dados JSON para projetos e tarefas
   - Persistência usando localStorage
   - Exportação/importação de projetos em formato JSON
   - Backup automático de dados

4. Interação:
   - Drag-and-drop para mover tarefas entre status
   - Edição inline de campos
   - Modais para criação/edição detalhada
   - Notificações para prazos próximos
   - Confirmação para ações irreversíveis

5. Recursos adicionais:
   - Gráfico de Gantt para visualização de cronograma
   - Temporizador pomodoro integrado para tarefas
   - Comentários e anexos em tarefas
   - Temas claro/escuro
   - Estatísticas de produtividade por projeto
```

## Calendário e Organizador de Eventos

```
Crie um calendário e organizador de eventos com as seguintes especificações:

1. Funcionalidade:
   - Visualizar calendário em diferentes formatos (dia, semana, mês)
   - Adicionar eventos com título, descrição, local, data/hora de início e fim
   - Configurar eventos recorrentes (diários, semanais, mensais, anuais)
   - Categorizar eventos por tipo (trabalho, pessoal, saúde, etc.)
   - Definir lembretes para eventos
   - Convidar participantes para eventos (simulado)
   - Pesquisar eventos por título, categoria ou data

2. Design:
   - Interface limpa e intuitiva com foco na visualização temporal
   - Esquema de cores: azul claro (#bbdefb) para fundo do calendário, cores distintas para diferentes categorias de eventos
   - Layout adaptável com barra lateral para detalhes de eventos
   - Visualização compacta em dispositivos móveis
   - Design inspirado no Google Calendar/Microsoft Outlook

3. Dados:
   - Estrutura de dados JSON para eventos e configurações
   - Persistência usando localStorage
   - Categorias pré-definidas com cores associadas
   - Exportação de eventos em formato iCalendar (.ics)

4. Interação:
   - Drag-and-drop para mover/redimensionar eventos
   - Clique rápido para visualizar detalhes, clique duplo para editar
   - Navegação intuitiva entre períodos (anterior/próximo)
   - Zoom para ajustar nível de detalhe na visualização
   - Atalhos de teclado para ações comuns

5. Recursos adicionais:
   - Visualização de previsão do tempo integrada
   - Modo de agenda para listagem cronológica de eventos
   - Sincronização com horário local e suporte a fusos horários
   - Temas claro/escuro
   - Estatísticas de ocupação (horas comprometidas por categoria)
```

## Aplicativo de Notas e Documentação

```
Crie um aplicativo de notas e documentação com as seguintes especificações:

1. Funcionalidade:
   - Criar e editar notas com formatação rica (negrito, itálico, listas, links)
   - Organizar notas em pastas e subpastas
   - Adicionar tags às notas para categorização
   - Pesquisar notas por conteúdo, título ou tags
   - Fixar notas importantes no topo
   - Exportar notas em diferentes formatos (texto, HTML, PDF)
   - Modo de edição e modo de visualização

2. Design:
   - Interface minimalista focada no conteúdo
   - Esquema de cores: fundo branco, elementos em tons de roxo (#673ab7), detalhes em cinza
   - Layout com três painéis: navegação de pastas, lista de notas, editor/visualizador
   - Tipografia clara e legível
   - Design responsivo que colapsa painéis em dispositivos menores
   - Inspirado em aplicativos como Notion/Evernote

3. Dados:
   - Estrutura de dados hierárquica para pastas e notas
   - Persistência usando localStorage
   - Salvamento automático durante edição
   - Versionamento básico de notas (histórico de alterações)

4. Interação:
   - Editor WYSIWYG com barra de ferramentas
   - Atalhos de teclado para formatação
   - Drag-and-drop para organizar notas e pastas
   - Modo de foco para escrita sem distrações
   - Feedback visual para ações (salvamento, exportação)

5. Recursos adicionais:
   - Modo escuro para leitura noturna
   - Suporte a código com syntax highlighting
   - Inserção de imagens e links
   - Estatísticas de escrita (contagem de palavras, tempo de leitura)
   - Modelos pré-definidos para diferentes tipos de notas
```
