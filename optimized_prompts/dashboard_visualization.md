# Prompt Otimizado: Dashboards e Visualização de Dados

## Dashboard de Vendas

```
Crie um dashboard de vendas interativo com as seguintes especificações:

1. Funcionalidade:
   - Exibir métricas-chave de vendas (receita total, número de vendas, ticket médio, taxa de conversão)
   - Mostrar gráficos de tendências de vendas ao longo do tempo (diário, semanal, mensal, anual)
   - Apresentar distribuição de vendas por categoria de produto
   - Exibir ranking dos produtos mais vendidos
   - Mostrar comparação de desempenho atual vs. período anterior
   - Permitir filtrar dados por período, categoria e região

2. Design:
   - Interface limpa e profissional com tema corporativo
   - Esquema de cores: azul (#1a73e8) para elementos principais, cinza claro (#f8f9fa) para fundo, verde (#34a853) para métricas positivas, vermelho (#ea4335) para métricas negativas
   - Layout em grid com cards para cada métrica/gráfico
   - Design responsivo que reorganiza elementos em telas menores
   - Inspirado no Google Analytics/Tableau

3. Dados:
   - Dados fictícios de vendas gerados automaticamente
   - Estrutura de dados JSON para simular API de vendas
   - Categorias de produtos: Eletrônicos, Roupas, Alimentos, Casa e Jardim, Esportes
   - Regiões: Norte, Sul, Leste, Oeste, Internacional

4. Interação:
   - Filtros interativos no topo do dashboard
   - Tooltips detalhados ao passar o mouse sobre elementos dos gráficos
   - Drill-down ao clicar em categorias para ver detalhes
   - Alternância entre diferentes visualizações (diário, semanal, mensal, anual)
   - Animações suaves para transições entre estados

5. Recursos adicionais:
   - Exportação de gráficos como imagens
   - Modo de apresentação em tela cheia
   - Alternância entre tema claro/escuro
   - Salvamento de configurações de filtros
   - Alertas visuais para métricas abaixo das metas
```

## Dashboard de Monitoramento de Desempenho

```
Crie um dashboard de monitoramento de desempenho de sistema com as seguintes especificações:

1. Funcionalidade:
   - Exibir métricas de desempenho em tempo real (CPU, memória, disco, rede)
   - Mostrar gráficos de linha para tendências de uso de recursos
   - Apresentar status de serviços (online/offline)
   - Exibir logs de eventos recentes com níveis de severidade
   - Mostrar tempo de atividade (uptime) e disponibilidade
   - Permitir ajustar intervalo de tempo para visualização (última hora, dia, semana)

2. Design:
   - Interface técnica e funcional com tema escuro
   - Esquema de cores: fundo escuro (#1e1e1e), texto claro (#f0f0f0), verde (#00b894) para status normal, amarelo (#fdcb6e) para avisos, vermelho (#d63031) para crítico
   - Layout em grid com painéis redimensionáveis
   - Gráficos minimalistas com foco na legibilidade
   - Design inspirado em ferramentas como Grafana/Prometheus

3. Dados:
   - Dados simulados de métricas de sistema gerados em intervalos regulares
   - Estrutura de dados JSON para simular API de monitoramento
   - Serviços fictícios: Web Server, Database, API, Cache, Authentication
   - Níveis de logs: Info, Warning, Error, Critical

4. Interação:
   - Atualização automática de dados a cada 5 segundos
   - Zoom em gráficos para análise detalhada
   - Filtros para tipos de eventos e níveis de severidade
   - Tooltips detalhados ao passar o mouse sobre pontos de dados
   - Notificações visuais para eventos críticos

5. Recursos adicionais:
   - Modo de alerta com destaque visual para métricas críticas
   - Comparação de métricas atuais com valores históricos
   - Configuração de limites para alertas
   - Exportação de dados para análise externa
   - Visualização de correlação entre diferentes métricas
```

## Dashboard de Análise Financeira

```
Crie um dashboard de análise financeira pessoal com as seguintes especificações:

1. Funcionalidade:
   - Exibir resumo financeiro (receitas, despesas, saldo, economia)
   - Mostrar gráfico de fluxo de caixa mensal
   - Apresentar distribuição de despesas por categoria
   - Exibir tendências de gastos ao longo do tempo
   - Mostrar previsão de saldo futuro baseado em padrões atuais
   - Permitir filtrar dados por período e categoria

2. Design:
   - Interface elegante e minimalista com tema claro
   - Esquema de cores: verde (#4caf50) para receitas, vermelho (#f44336) para despesas, azul (#2196f3) para elementos de interface
   - Layout organizado com seções claramente definidas
   - Visualizações de dados limpas e intuitivas
   - Design responsivo para desktop e mobile
   - Inspirado no Mint/YNAB

3. Dados:
   - Dados financeiros fictícios gerados para demonstração
   - Estrutura de dados JSON para transações
   - Categorias de despesas: Moradia, Alimentação, Transporte, Lazer, Saúde, Educação
   - Tipos de receitas: Salário, Investimentos, Freelance, Outros

4. Interação:
   - Filtros de data com seleção de intervalo personalizado
   - Drill-down em categorias para ver transações individuais
   - Comparação entre períodos (mês atual vs. anterior)
   - Tooltips detalhados em gráficos
   - Animações suaves para transições

5. Recursos adicionais:
   - Metas financeiras com indicadores de progresso
   - Alertas para gastos acima da média em categorias
   - Recomendações para economia baseadas em padrões de gastos
   - Exportação de relatórios em PDF
   - Simulador de cenários financeiros
```
